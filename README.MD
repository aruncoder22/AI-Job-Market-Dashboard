\# AI Job Market Dashboard (Power BI)



\##  Overview



This project analyzes the \*\*AI job market trends\*\* using Power BI.

It provides insights into job roles, salaries, skills demand, and hiring patterns.



\---



\## Key Insights



\* High demand for AI/ML roles

\* Top companies hiring in AI domain

\* Most required skills (Python, SQL, Power BI)

\* Salary distribution across roles



\---



\## Tools Used



\* Power BI

\* Excel

\* Python (for data cleaning)



\---



\##  Purpose



This project helps:



\* Job seekers understand market demand

\* Students learn data visualization

\* Recruiters analyze hiring trends
